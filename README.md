# cmd_interactive_nodejs
App de linha de comando interativa. 

# CRUD de Usuários
 - Usuário poderá realizar operações básicas do crud através do shell.
 
# Tecnologias aplicadas

- NodeJS
- CommanderJS
- InquirerJS
- MongoDB
- Yarn


**Crie um link simbólico com yarn**
```
cd pasta_do_projeto
yarn link
```
